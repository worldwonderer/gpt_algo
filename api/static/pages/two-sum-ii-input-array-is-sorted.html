两数之和 II - 输入有序数组 标签: 数组 双指针 二分查找    难度: Medium

Submission

class Solution:
    def twoSum(self, numbers: List[int], target: int) -> List[int]:
        lo = 0
        hi = len(numbers) - 1
        while lo < hi:
            s = numbers[lo] + numbers[hi]
            left = numbers[lo]
            right = numbers[hi]
            if s < target:
                while lo < hi and numbers[lo] == left:
                    lo += 1
            elif s > target:
                while lo < hi and numbers[hi] == right:
                    hi -= 1
            else:
                return [lo+1, hi+1]

运行时间: 40 ms

内存: 15.3 MB

Explain

这个题解使用双指针的方法在有序数组中查找两数之和等于目标值的一对数字。初始时左指针 lo 指向数组开头，右指针 hi 指向数组结尾。每次计算两个指针对应元素之和 s，如果 s 小于目标值，说明需要增大 lo 以获得更大的值；如果 s 大于目标值，说明需要减小 hi 以获得更小的值；如果 s 等于目标值，则找到了符合条件的一对数，返回它们的下标即可。在移动指针时，还跳过了重复的元素以优化效率。

时间复杂度: O(n)

空间复杂度: O(1)

class Solution:
    def twoSum(self, numbers: List[int], target: int) -> List[int]:
        lo = 0  # 左指针，初始指向数组开头
        hi = len(numbers) - 1  # 右指针，初始指向数组结尾
        while lo < hi:
            s = numbers[lo] + numbers[hi]  # 计算当前左右指针元素之和
            left = numbers[lo]  
            right = numbers[hi]
            if s < target:  # 若和小于目标值
                while lo < hi and numbers[lo] == left:  # 跳过左指针所指的重复元素
                    lo += 1
            elif s > target:  # 若和大于目标值
                while lo < hi and numbers[hi] == right:  # 跳过右指针所指的重复元素
                    hi -= 1
            else:  # 若和等于目标值
                return [lo+1, hi+1]  # 返回左右指针的下标（从1开始）
