class Solution:
    def calculate(self, s: str) -> int:
        x, y = 1, 0
        for i in s:
            if i == "A":
                x = 2 * x + y
            else:
                y = 2 * y + x
        return x + y

运行时间: 16 ms

内存: 16.5 MB

Explain

该题解通过模拟每一步的运算来解决问题。初始时，x和y分别被设置为1和0。然后，根据输入字符串s中的每一个字符，进行相应的'A'或'B'操作。如果是'A'，根据给定的操作规则更新x的值；如果是'B'，则更新y的值。这样，通过顺序地执行字符串s中的所有指令，最终得到x和y的值，并计算它们的和作为最终结果。

时间复杂度: O(n)

空间复杂度: O(1)

class Solution:
    def calculate(self, s: str) -> int:
        x, y = 1, 0  # 初始化x和y
        for i in s:  # 遍历字符串s
            if i == 'A':
                x = 2 * x + y  # 执行'A'操作
            else:
                y = 2 * y + x  # 执行'B'操作
        return x + y  # 返回x和y的和
