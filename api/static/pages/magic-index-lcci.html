魔术索引 标签: 数组 二分查找    难度: Easy

Submission

class Solution:
    def findMagicIndex(self, nums: List[int]) -> int:
        #最快的是2分
        #但是2分怎么找到最左边的那个？
        for a in range(0,len(nums)):
            if nums[a]==a:
                return a
        return -1

运行时间: 18 ms

内存: 16.9 MB

Explain

此题解采用了直接的线性搜索方法。从数组的第一个元素开始遍历，一旦发现某个索引i满足条件nums[i] == i，即判断为魔术索引，立即返回该索引。如果遍历完整个数组都没有找到符合条件的索引，则返回-1。

时间复杂度: O(n)

空间复杂度: O(1)

class Solution:
    def findMagicIndex(self, nums: List[int]) -> int:
        # 遍历数组的每个元素
        for index in range(0, len(nums)):
            # 如果当前元素值等于其索引，则找到魔术索引
            if nums[index] == index:
                return index
        # 如果没有找到任何魔术索引，返回-1
        return -1
