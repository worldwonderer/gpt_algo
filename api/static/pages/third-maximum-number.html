Explain

该题解的思路是先将原数组去重，然后判断去重后的数组长度：如果长度大于等于3，则返回排序后的数组的倒数第三个元素，即第三大的数；否则返回去重后数组的最大值。

时间复杂度: O(nlogn)

空间复杂度: O(n)

class Solution:
    def thirdMax(self, nums: List[int]) -> int:
        # 去重
        a = list(set(nums))
        # 判断去重后的数组长度
        if len(a) >= 3:
            # 如果长度大于等于3，返回排序后的倒数第三个元素
            return sorted(a)[-3]
        else:
            # 否则返回去重后数组的最大值
            return max(a)
