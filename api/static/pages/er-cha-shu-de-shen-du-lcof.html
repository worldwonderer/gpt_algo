计算二叉树的深度 标签: 树 深度优先搜索 广度优先搜索 二叉树    难度: Easy

Submission

# Definition for a binary tree node.
# class TreeNode:
#     def __init__(self, x):
#         self.val = x
#         self.left = None
#         self.right = None

class Solution:
    def maxDepth(self, root: TreeNode) -> int:
        if root is None:
            return 0
        return max(self.maxDepth(root.left), self.maxDepth(root.right)) + 1

运行时间: 56 ms

内存: 16.5 MB

Explain

该题解采用递归方法来计算二叉树的最大深度。它首先检查当前的树节点是否为 None，如果是，则当前深度为0。如果不是，它递归地计算左子树和右子树的最大深度，取两者中的较大值，然后加上当前的根节点（加1），最终返回这个值作为整棵树的最大深度。

时间复杂度: O(n)

空间复杂度: O(n)（最坏情况），O(log(n))（平均情况）

# Definition for a binary tree node.
# class TreeNode:
#     def __init__(self, x):
#         self.val = x
#         self.left = None
#         self.right = None

class Solution:
    def maxDepth(self, root: TreeNode) -> int:
        # 如果当前节点为空，则返回深度0
        if root is None:
            return 0
        # 递归地计算左子树和右子树的最大深度，然后取最大值，再加上当前根节点
        return max(self.maxDepth(root.left), self.maxDepth(root.right)) + 1
