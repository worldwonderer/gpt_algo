根据身高重建队列 标签: 树状数组 线段树 数组 排序    难度: Medium

Submission

class Solution:
    def reconstructQueue(self, people: List[List[int]]) -> List[List[int]]:
        people.sort(key = lambda x: (-x[0], x[1]))
        que = []

        for p in people:
            que.insert(p[1], p)
        return que

运行时间: 24 ms

内存: 16.5 MB

Explain

这个题解的思路是先对输入的人员数组 people 进行排序，排序规则是按照身高 h 降序，如果身高相等则按照 k 值升序。排序后，从前往后遍历排序后的数组，根据每个人的 k 值，将其插入到结果数组 que 中的第 k 个位置。最后得到的 que 数组即为重建后的队列顺序。

时间复杂度: O(n^2)

空间复杂度: O(n)

class Solution:
    def reconstructQueue(self, people: List[List[int]]) -> List[List[int]]:
        # 按照身高 h 降序，如果身高相等则按照 k 值升序排序
        people.sort(key = lambda x: (-x[0], x[1]))
        que = []

        # 遍历排序后的数组
        for p in people:
            # 根据 k 值将每个人插入到结果数组 que 中的第 k 个位置
            que.insert(p[1], p)
        
        return que
