猜数字大小 标签: 二分查找 交互    难度: Easy

# The guess API is already defined for you.
# @param num, your guess
# @return -1 if num is higher than the picked number
#          1 if num is lower than the picked number
#          otherwise return 0
# def guess(num: int) -> int:

class Solution:
    def guessNumber(self, n: int) -> int:
        left,right=1,n
        while left <=right:
            mid=(left+right)//2
            if guess(mid)==0:
                return mid
            elif guess(mid)==-1:
                right =mid-1
            else:
                left =mid+1

运行时间: 17 ms

内存: 15.9 MB

Explain

这道题使用二分查找的思路来猜数字。每次根据 guess 函数的返回值，缩小搜索范围，直到找到目标数字。如果 guess 返回 -1，说明猜的数字偏大，缩小右边界；如果返回 1，说明猜的数字偏小，缩小左边界；如果返回 0，说明猜中了，直接返回当前数字。

时间复杂度: O(log n)

空间复杂度: O(1)

# The guess API is already defined for you.
# @param num, your guess
# @return -1 if num is higher than the picked number
#          1 if num is lower than the picked number
#          otherwise return 0
# def guess(num: int) -> int:

class Solution:
    def guessNumber(self, n: int) -> int:
        left, right = 1, n
        while left <= right:
            mid = (left + right) // 2
            if guess(mid) == 0:
                # 猜中了，直接返回当前数字
                return mid
            elif guess(mid) == -1:
                # 猜的数字偏大，缩小右边界 
                right = mid - 1
            else:
                # 猜的数字偏小，缩小左边界
                left = mid + 1
