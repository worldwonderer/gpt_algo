每棵子树内缺失的最小基因值 标签: 树 深度优先搜索 并查集 动态规划    难度: Hard

Submission

class Solution:
    def smallestMissingValueSubtree(self, parents: List[int], nums: List[int]) -> List[int]:
        n = len(nums)
        if min(nums)!=1: return [1]*n
        idx = nums.index(1)
        g = [[] for _ in range(n)]
        for i in range(1, n):
            g[parents[i]].append(i)
        ans = [1] * n
        vis = set()
        mn = 2
        son = -1
        while idx >= 0:
            vis.add(nums[idx])
            st = [i for i in g[idx] if i!=son]
            while st:
                cur = st.pop()
                vis.add(nums[cur]) 
                st.extend(g[cur])
            while mn in vis: mn += 1
            son = idx
            ans[idx] = mn
            idx = parents[idx]
        return ans

运行时间: 252 ms

内存: 50.1 MB

Explain

此题解采用的是一种高效的方法，首先检查基因值是否包含1，因为如果不包含1，则所有子树的最小缺失基因值必定是1。如果包含1，我们需要找到包含基因值1的节点，并采用从该节点向根节点追溯的方式，逐一查找每个节点所在子树的最小缺失基因值。使用DFS (深度优先搜索) 遍历节点，将访问过的基因值存入集合中，并实时更新未见的最小基因值。这种方法只需要处理包含1的节点到根节点路径上的子树，其他节点的答案直接是1，因此效率较高。

时间复杂度: O(n)

空间复杂度: O(n)

class Solution:
    def smallestMissingValueSubtree(self, parents: List[int], nums: List[int]) -> List[int]:
        n = len(nums)
        # 如果没有基因值1，则所有子树的最小缺失值为1
        if min(nums) != 1: return [1] * n
        # 找到基因值为1的节点的索引
        idx = nums.index(1)
        # 构建图，用于存储树中的父子关系
        g = [[] for _ in range(n)]
        for i in range(1, n):
            g[parents[i]].append(i)
        # 初始化答案数组，所有值设为1（最小可能缺失的基因值）
        ans = [1] * n
        # 创建一个集合来存储访问过的基因值
        vis = set()
        # 初始化最小缺失值为2（因为1已存在）
        mn = 2
        son = -1
        # 从包含基因值1的节点开始，向上遍历到根节点
        while idx >= 0:
            vis.add(nums[idx])
            # 获取当前节点的所有子节点，除了来时的节点
            st = [i for i in g[idx] if i != son]
            # DFS遍历子树，收集所有基因值
            while st:
                cur = st.pop()
                vis.add(nums[cur])
                st.extend(g[cur])
            # 更新最小缺失的基因值
            while mn in vis: mn += 1
            # 向上移动到父节点，继续检查
            son = idx
            ans[idx] = mn
            idx = parents[idx]
        return ans
