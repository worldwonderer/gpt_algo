两数相加 II 标签: 栈 链表 数学    难度: Medium

Submission

# Definition for singly-linked list.
# class ListNode:
#     def __init__(self, x):
#         self.val = x
#         self.next = None

class Solution:
    def addTwoNumbers(self, l1: ListNode, l2: ListNode) -> ListNode:
        s1 = []
        while l1:
            s1.append(l1.val)
            l1 = l1.next
    
        s2 = []
        while l2:
            s2.append(l2.val)
            l2 = l2.next
        
        carry = 0
        head = None
        while s1 or s2 or carry == 1:
            total = 0
            total += s1.pop() if s1 else 0
            total += s2.pop() if s2 else 0
            if carry == 1:
                total += 1
                carry = 0
            if total >= 10:
                carry = 1
                total -= 10
            node = ListNode(total)
            node.next = head
            head = node
        return head

运行时间: 72 ms

内存: 14.9 MB

Explain

这道题的思路是利用栈来模拟数字的加法过程。由于链表的最高位在链表的开始位置，我们可以使用栈将链表中的数字按顺序存储，这样栈顶的元素就是数字的最低位。接着我们可以像手动计算两个数字之和那样，从栈顶开始逐位相加，并处理进位。最后，我们将结果构造成一个新的链表并返回。

时间复杂度: O(m+n)

空间复杂度: O(m+n)
