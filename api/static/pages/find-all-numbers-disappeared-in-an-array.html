找到所有数组中消失的数字 标签: 数组 哈希表    难度: Easy

Submission

class Solution:
    def findDisappearedNumbers(self, nums: List[int]) -> List[int]:
        res = []

        setNums = set(nums)
        for i in range(1, len(nums) + 1):
            if i not in setNums:
                res.append(i)

        return res

运行时间: 38 ms

内存: 25.2 MB

Explain

该题解的思路是先将原数组转换为集合，然后遍历从1到n的所有数字，检查每个数字是否在集合中。如果不在集合中，则说明该数字在原数组中消失，将其添加到结果数组中。最后返回结果数组。

时间复杂度: O(n)

空间复杂度: O(n)

class Solution:
    def findDisappearedNumbers(self, nums: List[int]) -> List[int]:
        res = []

        # 将原数组转换为集合
        setNums = set(nums)
        
        # 遍历从1到n的所有数字
        for i in range(1, len(nums) + 1):
            # 检查当前数字是否在集合中
            if i not in setNums:
                # 如果不在集合中，说明该数字在原数组中消失，将其添加到结果数组中
                res.append(i)

        return res
