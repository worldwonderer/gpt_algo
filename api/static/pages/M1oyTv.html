最小覆盖子串 标签: 哈希表 字符串 滑动窗口    难度: Hard

class Solution:
    def minWindow(self, s: str, t: str) -> str:
        ###滑动窗口+双指针+计数
        dic=collections.defaultdict(int)
        for tt in t:
            dic[tt]+=1
        cnt=len(t)
        res=(0,len(s))
        i,j=0,0
        while j<len(s):
            if dic[s[j]]>0:
                cnt-=1
            dic[s[j]]-=1
            if cnt==0:
                while dic[s[i]]<0:
                    dic[s[i]]+=1
                    i+=1
                if j-i<res[1]-res[0]:res=(i,j)
                dic[s[i]]+=1
                i+=1
                cnt+=1
            j+=1
        return s[res[0]:res[1]+1] if res[1]!=len(s) else ''

运行时间: 56 ms

内存: 16.3 MB

Explain

这道题使用了滑动窗口和双指针的方法。首先，使用一个字典记录字符串t中每个字符的数量。然后，使用两个指针i和j来表示当前的滑动窗口，初始时都指向s的开头。随着j的向右移动，如果当前字符在t中，就减少字典中对应字符的数量，并且减少计数器cnt的值。当cnt变为0时，说明当前窗口已经包含了t中的所有字符。接着，尝试移动左指针i，直到窗口不再包含t中的所有字符。在这个过程中，记录最小的窗口大小，并更新结果。最后，返回最小覆盖子串。

时间复杂度: O(n)

空间复杂度: O(1)

class Solution:
    def minWindow(self, s: str, t: str) -> str:
        # 滑动窗口+双指针+计数
        dic = collections.defaultdict(int)
        for tt in t:
            dic[tt] += 1
        cnt = len(t)
        res = (0, len(s))
        i, j = 0, 0
        while j < len(s):
            if dic[s[j]] > 0:
                cnt -= 1
            dic[s[j]] -= 1
            if cnt == 0:
                while dic[s[i]] < 0:
                    dic[s[i]] += 1
                    i += 1
                if j - i < res[1] - res[0]:
                    res = (i, j)
                dic[s[i]] += 1
                i += 1
                cnt += 1
            j += 1
        return s[res[0]:res[1] + 1] if res[1] != len(s) else ''
