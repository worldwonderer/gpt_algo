Explain

此题解采用了和的差值法。首先计算出0到n的和s，然后遍历数组nums，将数组中每个元素的值从s中减去。最后，s中剩余的值就是缺失的数字。

时间复杂度: O(n)

空间复杂度: O(1)

class Solution:
    def missingNumber(self, nums: List[int]) -> int:
        # 计算0到n的和
        n = len(nums)
        s = (1 + n) * n // 2
        # 遍历数组，更新和
        for i in range(n):
            s -= nums[i]
        # 返回缺失的数字
        return s
