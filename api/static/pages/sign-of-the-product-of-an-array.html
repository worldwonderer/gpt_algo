数组元素积的符号 标签: 数组 数学    难度: Easy

class Solution:
    def arraySign(self, nums: List[int]) -> int:
        if 0 in nums:
            return 0
        c=[]
        for x in nums:
            if x<0:
                c.append(x)
        if len(c)%2==0:
            return 1
        else:   
            return -1

运行时间: 20 ms

内存: 16.2 MB

Explain

此题解的核心思路是通过统计数组中负数的个数和是否包含零来决定乘积的符号。首先，如果数组中包含零，则直接返回0，因为任何数与0相乘都为0。如果数组中没有零，那么题解通过一个循环遍历数组，记录所有负数的个数。最后，根据负数的个数是奇数还是偶数来决定返回1或-1。如果负数个数为偶数，则所有负数的乘积为正，反之为负。

时间复杂度: O(n)

空间复杂度: O(n)

# 定义Solution类
class Solution:
    # arraySign函数定义
    def arraySign(self, nums: List[int]) -> int:
        # 检查数组中是否有0，有则直接返回0
        if 0 in nums:
            return 0
        # 初始化负数计数器
        neg_count = 0
        # 遍历数组元素
        for x in nums:
            # 如果元素小于0，增加负数计数器
            if x < 0:
                neg_count += 1
        # 根据负数的个数的奇偶性返回符号
        if neg_count % 2 == 0:
            return 1
        else:
            return -1
