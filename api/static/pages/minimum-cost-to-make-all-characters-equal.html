使所有字符相等的最小成本 标签: 贪心 字符串 动态规划    难度: Medium

Submission

class Solution:
    def minimumCost(self, s: str) -> int:
        if len(s) == 1:
            return 0
        # 分为左右两部分，如果长度为奇数，中间部分特殊处理
        n = len(s)
        mid = n >> 1
        ans = 0
        for sub in s[:mid], s[-1:-mid-1:-1]:
            for i in range(1, mid):
                if sub[i] != sub[i - 1]:
                    ans += i

        if s[mid] != s[mid - 1]:
            ans += mid
        if n & 1 and s[mid] != s[mid + 1]:
            ans += mid

        return ans

运行时间: 94 ms

内存: 16.5 MB

Explain

此题解的思路是将字符串分为左右两部分进行处理。对于每一部分，遍历字符串，比较相邻字符是否相等。如果不相等，就累加当前下标作为成本。对于长度为奇数的字符串，中间的字符需要特殊处理。最后将所有成本累加得到最小成本。

时间复杂度: O(n)

空间复杂度: O(1)
