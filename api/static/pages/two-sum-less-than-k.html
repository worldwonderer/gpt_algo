小于 K 的两数之和 标签: 数组 双指针 二分查找 排序    难度: Easy

Submission

class Solution:
    def twoSumLessThanK(self, nums: List[int], k: int) -> int:
        nums.sort()

        ans = -1
        left = 0
        right = len(nums) - 1
        while left < right:
            s = nums[left] + nums[right]
            if s < k:
                ans = max(ans, s)
                left += 1
            else:
                right -= 1
        return ans

运行时间: 23 ms

内存: 16.1 MB

Explain

该题解采用双指针方法。首先对数组进行排序，然后使用两个指针分别指向数组的起始位置和结束位置。在遍历过程中，计算两个指针指向的元素之和，如果和小于k，则更新答案，并将左指针向右移动一位；如果和大于等于k，则将右指针向左移动一位。这样可以在保证和小于k的前提下，尽可能地让和大，直到两个指针相遇为止。

时间复杂度: O(n log n)

空间复杂度: O(n)

class Solution:
    def twoSumLessThanK(self, nums: List[int], k: int) -> int:
        nums.sort()  # 对数组进行排序

        ans = -1  # 初始化答案为-1
        left = 0  # 左指针初始化为数组起始位置
        right = len(nums) - 1  # 右指针初始化为数组结束位置
        while left < right:  # 当左指针小于右指针时，进行遍历
            s = nums[left] + nums[right]  # 计算两个指针指向的元素之和
            if s < k:  # 如果和小于k
                ans = max(ans, s)  # 更新答案
                left += 1  # 将左指针向右移动一位
            else:
                right -= 1  # 否则，将右指针向左移动一位
        return ans  # 返回答案
