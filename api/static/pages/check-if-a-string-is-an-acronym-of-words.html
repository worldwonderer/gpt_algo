判别首字母缩略词 标签: 数组 字符串    难度: Easy

Submission

class Solution:
    def isAcronym(self, words: List[str], s: str) -> bool:
        if len(words) != len(s): return False
        for i, e in enumerate(words):
            if e[0] != s[i]:
                return False
        else:
            return True

运行时间: 16 ms

内存: 16.5 MB

Explain

该题解首先检查输入字符串数组 `words` 的长度是否与字符串 `s` 的长度相等，因为只有当两者长度相同时，才有可能通过 `words` 的每个单词的第一个字符完全匹配字符串 `s`。接下来，题解通过遍历 `words`，对于每个单词，检查它的第一个字符是否与 `s` 中对应位置的字符相同。如果所有字符都匹配，则返回 `true`；否则，在发现第一个不匹配的字符时立即返回 `false`。

时间复杂度: O(n)

空间复杂度: O(1)

# 定义Solution类

class Solution:
    # 定义isAcronym方法，接收words列表和字符串s
    def isAcronym(self, words: List[str], s: str) -> bool:
        # 如果words的长度不等于s的长度，直接返回False
        if len(words) != len(s): return False
        # 遍历words，检查每个单词的首字母是否与s中对应位置的字符相同
        for i, e in enumerate(words):
            if e[0] != s[i]:  # 如果不相同，返回False
                return False
        # 如果所有首字母都匹配，返回True
        else:
            return True
