买卖股票的最佳时机 标签: 数组 动态规划    难度: Easy

Submission

class Solution:
    def maxProfit(self, prices: List[int]) -> int:
        n = len(prices)
        dp = [[0, 1] for _ in range(n+1)]
        dp[0][0] = 0
        dp[0][1] = float('-inf')
        for i in range(1, n+1):
            dp[i][0] = max(dp[i-1][0], dp[i-1][1]+prices[i-1])
            dp[i][1] = max(dp[i-1][1], -prices[i-1])
        return dp[-1][0]

运行时间: 792 ms

内存: 29.5 MB

Explain

这个题解使用了动态规划的方法。它维护了一个二维的 dp 数组，其中 dp[i][0] 表示第 i 天结束时不持有股票的最大利润，dp[i][1] 表示第 i 天结束时持有股票的最大利润。对于每一天，我们有两种选择：1. 不持有股票，此时最大利润为前一天不持有股票的利润和前一天持有股票然后今天卖出的利润中的较大值；2. 持有股票，此时最大利润为前一天持有股票的利润和前一天不持有股票然后今天买入的利润中的较大值。最终的结果是最后一天不持有股票的最大利润。

时间复杂度: O(n)

空间复杂度: O(n)

class Solution:
    def maxProfit(self, prices: List[int]) -> int:
        n = len(prices)
        # 初始化 dp 数组
        dp = [[0, 1] for _ in range(n+1)]
        dp[0][0] = 0
        dp[0][1] = float('-inf')
        # 遍历 prices 数组
        for i in range(1, n+1):
            # 计算第 i 天不持有股票的最大利润
            dp[i][0] = max(dp[i-1][0], dp[i-1][1]+prices[i-1])
            # 计算第 i 天持有股票的最大利润
            dp[i][1] = max(dp[i-1][1], -prices[i-1])
        # 返回最后一天不持有股票的最大利润
        return dp[-1][0]
