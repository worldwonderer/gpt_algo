三维形体的表面积 标签: 几何 数组 数学 矩阵    难度: Easy

class Solution:
    def surfaceArea(self, grid: List[List[int]]) -> int:
        # 派一个小人横向走一遍+纵向走一遍，上下台阶求费力值，
        # 数组大于零的个数*2即为上下底面积
        a = 0   # 计算大于零的数即上下底面积
        ans = 0 # 计算侧面积
        n = len(grid)   # 数组的横竖个数
        for i in range(n):
            row = 0
            lie = 0
            for j in range(n):
                if grid[i][j] > 0:
                    a += 1      # 大于零说明地面有方格+1
                # 走台阶费力值就是上一次台阶和本次台阶的差值
                ans = ans + abs(row - grid[i][j]) + abs(lie - grid[j][i])
                lie = grid[j][i]
                row = grid[i][j]
            ans = ans + lie + row
        return (ans+a*2)    # 上下底面加侧面

运行时间: 38 ms

内存: 16.0 MB

Explain

此题解首先遍历网格中的每个单元格，计算每个单元格上正方体的底面和顶面总面积。如果一个单元格中的正方体数量大于0，则它有底面和顶面，因此为每个正方体增加2的面积。对于侧面积，考虑每个单元格在横向和纵向与其相邻单元格之间的正方体数量差值。通过计算这些差值的绝对值，可以得到正方体间未被相邻正方体覆盖的侧面面积。遍历完成后，将底面和顶面的总面积与侧面面积加和，得到最终的结果。

时间复杂度: O(n^2)

空间复杂度: O(1)

class Solution:
    def surfaceArea(self, grid: List[List[int]]) -> int:
        a = 0   # 存储正方体的顶面和底面总面积
        ans = 0 # 存储所有侧面的总面积
        n = len(grid)   # 网格的大小
        for i in range(n):
            row = 0
            lie = 0
            for j in range(n):
                if grid[i][j] > 0:
                    a += 1      # 计算具有顶面和底面的正方体数量
                # 计算当前正方体与横向和纵向相邻正方体的高度差
                ans = ans + abs(row - grid[i][j]) + abs(lie - grid[j][i])
                lie = grid[j][i]
                row = grid[i][j]
            # 在每一行和每一列结束时加上最后一个正方体的高度，因为最外侧的正方体也有侧面
            ans = ans + lie + row
        return (ans + a * 2)    # 计算总表面积，包括侧面和顶面底面
