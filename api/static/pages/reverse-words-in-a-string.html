反转字符串中的单词

标签: 双指针 字符串

难度: Medium

给你一个字符串 s ，请你反转字符串中 单词 的顺序。

单词 是由非空格字符组成的字符串。s 中使用至少一个空格将字符串中的 单词 分隔开。

返回 单词 顺序颠倒且 单词 之间用单个空格连接的结果字符串。

注意：输入字符串 s中可能会存在前导空格、尾随空格或者单词间的多个空格。返回的结果字符串中，单词间应当仅用单个空格分隔，且不包含任何额外的空格。

示例 1：

输入：s = "the sky is blue"
输出："blue is sky the"

示例 2：

输入：s = "  hello world  "
输出："world hello"
解释：反转后的字符串中不能存在前导空格和尾随空格。

示例 3：

输入：s = "a good   example"
输出："example good a"
解释：如果两个单词间有多余的空格，反转后的字符串需要将单词间的空格减少到仅有一个。

提示：

  • 1 <= s.length <= 104
  • s 包含英文大小写字母、数字和空格 ' '
  • s 中 至少存在一个 单词

进阶：如果字符串在你使用的编程语言中是一种可变数据类型，请尝试使用 O(1) 额外空间复杂度的 原地 解法。

Submission

运行时间: 15 ms

内存: 16.1 MB

class Solution:
    def reverseWords(self, s: str) -> str:
        # 删除前后空白
        s = s.strip()
        # 反转整个字符串
        s = s[::-1]
        # 将字符串拆分为单词，并反转每个单词
        s = ' '.join(word[::-1] for word in s.split())
        return s

Explain

该题解的思路是：首先去除字符串两端的空白，然后反转整个字符串，最后将字符串按空格拆分成单词数组，对每个单词进行反转，并用空格连接成最终结果字符串。

时间复杂度: O(n)

空间复杂度: O(n)

class Solution:
    def reverseWords(self, s: str) -> str:
        # 删除前后空白
        s = s.strip()
        # 反转整个字符串
        s = s[::-1]
        # 将字符串拆分为单词，并反转每个单词
        s = ' '.join(word[::-1] for word in s.split())
        return s

Explore

这种做法是为了将单词的顺序反转到最终需要的顺序。在原字符串中，单词是按正常阅读顺序排列的，例如在字符串 'hello world' 中，'hello' 在 'world' 前面。通过首先反转整个字符串，我们得到 'dlrow olleh'，这时单词间的顺序已经被反转，即原来的最后一个单词 'world' 变为第一个。接下来只需要将每个单词内部的字符顺序再次反转，就能得到正确的单词顺序和字符顺序。

使用 `split()` 方法可以非常高效地处理字符串中的所有类型空格（包括多个连续空格和不同类型的空白字符）。当调用 `split()` 时不带参数，默认会将字符串拆分为单词数组，自动忽略任何形式的空白字符（包括空格、制表符、换行符等）。这样就避免了复杂的手动处理，简化了代码，并保持了高效的处理效率。其他方法，如手动遍历字符串处理空格，通常会更复杂、更易出错，并且效率较低。

列表推导式是一种简洁且通常效率较高的方法来创建新列表，特别是在涉及简单操作（如字符串反转）时。在处理大型字符串时，虽然列表推导式提供了很好的性能，但需要注意每个单词反转生成的是新字符串，这会导致额外的内存使用。总体来说，对于大多数实际应用，这种方法的性能是可接受的。然而，如果处理极大的数据或性能要求极高的应用，可能需要考虑更优化的方法，如使用更低级的操作来减少内存占用和提高效率。

通过使用 `split()` 方法，在这种情况下非常有效，因为它默认将字符串拆分为单词列表，并且自动忽略字符串中的所有额外空格，包括单词之间的多个空格。在单词被反转并重新组装成字符串时，使用 `' '.join()` 方法可以保证单词之间只有一个空格。这样，即使原字符串中单词之间有多个空格，返回的字符串也只会含有单词之间的一个空格，从而避免了任何额外的空格。
