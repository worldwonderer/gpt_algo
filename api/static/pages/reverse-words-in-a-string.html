反转字符串中的单词 标签: 双指针 字符串    难度: Medium

Submission

class Solution:
    def reverseWords(self, s: str) -> str:
        # 删除前后空白
        s = s.strip()
        # 反转整个字符串
        s = s[::-1]
        # 将字符串拆分为单词，并反转每个单词
        s = ' '.join(word[::-1] for word in s.split())
        return s

运行时间: 15 ms

内存: 16.1 MB

Explain

该题解的思路是：首先去除字符串两端的空白，然后反转整个字符串，最后将字符串按空格拆分成单词数组，对每个单词进行反转，并用空格连接成最终结果字符串。

时间复杂度: O(n)

空间复杂度: O(n)

class Solution:
    def reverseWords(self, s: str) -> str:
        # 删除前后空白
        s = s.strip()
        # 反转整个字符串
        s = s[::-1]
        # 将字符串拆分为单词，并反转每个单词
        s = ' '.join(word[::-1] for word in s.split())
        return s
