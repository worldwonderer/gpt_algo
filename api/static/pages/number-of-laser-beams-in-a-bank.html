银行中的激光束数量 标签: 数组 数学 字符串 矩阵    难度: Medium

Submission

class Solution:
    def numberOfBeams(self, bank: List[str]) -> int:
        ans = 0
        prevCnt = 0
        for row in bank:
            if '1' not in row:
                continue
            cnt = row.count('1')
            ans += prevCnt * cnt
            prevCnt = cnt
        
        return ans

运行时间: 28 ms

内存: 18.4 MB

Explain

这个题解的核心思想是利用一次遍历来计算行与行之间可能形成的激光束数量。对于每一行，首先检查该行是否包含安全设备（即包含'1'）。如果这一行有安全设备，计算这一行的设备数量。然后，将这个数量与前一次记录的行的设备数量相乘，得到的结果就是这两行之间的激光束数量。将这个数量累加到答案中，并更新前一行设备数量的记录。这样，通过一次遍历即可计算出所有可能的激光束数量。

时间复杂度: O(m*n)

空间复杂度: O(1)

class Solution:
    def numberOfBeams(self, bank: List[str]) -> int:
        ans = 0  # 初始化激光束总数
        prevCnt = 0  # 上一行有设备的行的设备数量
        for row in bank:  # 遍历每一行
            if '1' not in row:  # 如果这一行没有设备，跳过
                continue
            cnt = row.count('1')  # 计算当前行的设备数量
            ans += prevCnt * cnt  # 当前行和上一有设备行之间的激光束数量
            prevCnt = cnt  # 更新上一有设备行为当前行
        
        return ans  # 返回激光束的总数量
