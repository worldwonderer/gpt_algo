最小操作次数使数组元素相等 II 标签: 数组 数学 排序    难度: Medium

Submission

class Solution:
    def minMoves2(self, nums: List[int]) -> int:
        n = len(nums)
        nums.sort()
        a = nums[n//2]
        result = 0
        for i in nums:
            result += abs(i-a)
        return result

运行时间: 28 ms

内存: 17.0 MB

Explain

这个题解的思路是先将数组排序，然后找到中位数（如果数组长度为奇数则取中间的数，如果为偶数则取最中间两个数的任意一个）。接下来，计算所有数组元素到中位数的绝对差之和，即为最小操作数。

时间复杂度: O(nlogn)

空间复杂度: O(n)

class Solution:
    def minMoves2(self, nums: List[int]) -> int:
        n = len(nums)
        nums.sort()  # 对数组进行排序
        a = nums[n//2]  # 取中位数
        result = 0
        for i in nums:
            result += abs(i-a)  # 计算每个元素与中位数的绝对差，并累加
        return result  # 返回最小操作数
