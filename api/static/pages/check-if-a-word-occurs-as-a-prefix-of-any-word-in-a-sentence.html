检查单词是否为句中其他单词的前缀 标签: 双指针 字符串 字符串匹配    难度: Easy

Submission

class Solution:
    def isPrefixOfWord(self, sentence: str, searchWord: str) -> int:
        n = len(searchWord)
        temp = sentence.split()
        for i in range(len(temp)):
            if searchWord in temp[i] and temp[i][:n] == searchWord:
                return i + 1
        return -1

运行时间: 22 ms

内存: 16.0 MB

Explain

该题解采用的主要思路是首先将句子按空格分割成单词数组，然后遍历每个单词，检查给定的搜索词 `searchWord` 是否是当前单词的前缀。具体来说，通过判断 `searchWord` 是否等于单词从开始到 `searchWord` 长度的子串。如果是，立即返回当前单词的索引（从1开始计数）。如果遍历完所有单词都没有找到，返回-1。

时间复杂度: O(n)

空间复杂度: O(n)

class Solution:
    def isPrefixOfWord(self, sentence: str, searchWord: str) -> int:
        n = len(searchWord)  # 获取搜索词的长度
        temp = sentence.split()  # 将句子分割成单词列表
        for i in range(len(temp)):  # 遍历每个单词
            if searchWord in temp[i] and temp[i][:n] == searchWord:  # 检查searchWord是否在当前单词中，并且是否为前缀
                return i + 1  # 如果是，返回当前单词的索引（从1开始计数）
        return -1  # 如果没有找到任何匹配的前缀，返回-1
