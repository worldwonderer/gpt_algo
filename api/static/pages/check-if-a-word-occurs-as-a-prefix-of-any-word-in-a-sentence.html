检查单词是否为句中其他单词的前缀

标签: 双指针 字符串 字符串匹配

难度: Easy

给你一个字符串 sentence 作为句子并指定检索词为 searchWord ，其中句子由若干用 单个空格 分隔的单词组成。请你检查检索词 searchWord 是否为句子 sentence 中任意单词的前缀。

如果 searchWord 是某一个单词的前缀，则返回句子 sentence 中该单词所对应的下标（下标从 1 开始）。如果 searchWord 是多个单词的前缀，则返回匹配的第一个单词的下标（最小下标）。如果 searchWord 不是任何单词的前缀，则返回 -1 。

字符串 s 的 前缀 是 s 的任何前导连续子字符串。

示例 1：

输入：sentence = "i love eating burger", searchWord = "burg"
输出：4
解释："burg" 是 "burger" 的前缀，而 "burger" 是句子中第 4 个单词。

示例 2：

输入：sentence = "this problem is an easy problem", searchWord = "pro"
输出：2
解释："pro" 是 "problem" 的前缀，而 "problem" 是句子中第 2 个也是第 6 个单词，但是应该返回最小下标 2 。

示例 3：

输入：sentence = "i am tired", searchWord = "you"
输出：-1
解释："you" 不是句子中任何单词的前缀。

提示：

  • 1 <= sentence.length <= 100
  • 1 <= searchWord.length <= 10
  • sentence 由小写英文字母和空格组成。
  • searchWord 由小写英文字母组成。

Submission

运行时间: 22 ms

内存: 16.0 MB

class Solution:
    def isPrefixOfWord(self, sentence: str, searchWord: str) -> int:
        n = len(searchWord)
        temp = sentence.split()
        for i in range(len(temp)):
            if searchWord in temp[i] and temp[i][:n] == searchWord:
                return i + 1
        return -1

Explain

该题解采用的主要思路是首先将句子按空格分割成单词数组，然后遍历每个单词，检查给定的搜索词 `searchWord` 是否是当前单词的前缀。具体来说，通过判断 `searchWord` 是否等于单词从开始到 `searchWord` 长度的子串。如果是，立即返回当前单词的索引（从1开始计数）。如果遍历完所有单词都没有找到，返回-1。

时间复杂度: O(n)

空间复杂度: O(n)

class Solution:
    def isPrefixOfWord(self, sentence: str, searchWord: str) -> int:
        n = len(searchWord)  # 获取搜索词的长度
        temp = sentence.split()  # 将句子分割成单词列表
        for i in range(len(temp)):  # 遍历每个单词
            if searchWord in temp[i] and temp[i][:n] == searchWord:  # 检查searchWord是否在当前单词中，并且是否为前缀
                return i + 1  # 如果是，返回当前单词的索引（从1开始计数）
        return -1  # 如果没有找到任何匹配的前缀，返回-1

Explore

在算法中使用`searchWord in temp[i]`的条件主要是为了确保在进行前缀比较之前，当前单词确实包含`searchWord`作为子串。这个条件并不是性能优化的直接结果，而是减少不必要的字符串比较——如果`searchWord`不在当前单词中，那么它肯定不是前缀，所以可以直接跳过这个单词。虽然这样做确实可以在某些情况下提高效率，但主要目的是减少无效的操作和提高代码的逻辑清晰度。

如果`searchWord`为空字符串，根据算法逻辑，空字符串是任何字符串的前缀，因此算法会返回第一个单词的索引，即1。在实际应用中，是否需要对空字符串作为搜索词进行特殊处理取决于具体需求。如果要避免这种情况，可以在算法开始时添加一条检查，如果`searchWord`为空，则直接返回-1或者其他特定值，以表明搜索词不应为空。

是的，算法能够正确返回第一个匹配单词的索引。算法遍历单词数组，并且在找到第一个满足前缀条件的单词时就会停止搜索并返回当前单词的索引。因此，即使句子中包含重复单词，返回的仍将是第一个匹配的单词的索引。

如果句子`sentence`为空或只包含空格，`sentence.split()`会返回一个空列表。因此，遍历单词的循环将不会执行，算法最终将返回-1。这种处理是合理的，因为没有单词可供检查前缀。为了更加明确，可以在算法开始时添加对`sentence`内容的检查，如果`sentence`为空或全为空格，直接返回-1，这样可以更早地处理边界情况，避免进行不必要的操作，提高算法的效率和可读性。
