分割数组的方案数 标签: 数组 前缀和    难度: Medium

Submission

class Solution:
    def waysToSplitArray(self, nums: List[int]) -> int:
        a = list(accumulate(nums))
        p = a.pop() / 2
        return sum(i >= p for i in a)

运行时间: 70 ms

内存: 31.1 MB

Explain

此题解使用了前缀和的方法来简化数组分割点的计算。首先，使用Python内置的accumulate函数计算数组nums的前缀和，这样每个元素a[i]代表从nums[0]到nums[i]的元素和。接着，计算整个数组的总和并除以2，得到p，这是因为如果前半部分和大于等于后半部分和，那么前半部分和至少应该达到总和的一半。最后，遍历前缀和数组，计算有多少个元素的值大于等于p，这些元素对应的索引就是合法分割点的位置。

时间复杂度: O(n)

空间复杂度: O(n)

# Definition of the Solution class with a method to calculate the number of valid splits in the array

class Solution:
    def waysToSplitArray(self, nums: List[int]) -> int:
        # 计算数组的前缀和
        a = list(accumulate(nums))
        # 计算总和的一半，用于比较
        p = a.pop() / 2
        # 统计前缀和中大于等于总和一半的元素数量，即为合法分割的方案数
        return sum(i >= p for i in a)
