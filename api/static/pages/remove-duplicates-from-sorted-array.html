删除有序数组中的重复项 标签: 数组 双指针    难度: Easy

Submission

class Solution:
    def removeDuplicates(self, nums: List[int]) -> int:
        n = len(nums)
        i = 0
        for j in range(n):
            if nums[j] != nums[i]:
                i += 1
                nums[i] = nums[j]
        return i + 1

运行时间: 48 ms

内存: 15.7 MB

Explain

这个题解使用了双指针的思路。指针i用于记录当前已处理的不重复元素的位置，指针j用于遍历整个数组。当遇到与nums[i]不同的元素时，将其赋值给nums[i+1]，并将i后移一位。这样可以原地修改数组，将不重复的元素移动到数组的前面，同时保持它们的相对顺序。最后，i+1就是删除重复元素后数组的新长度。

时间复杂度: O(n)

空间复杂度: O(1)

class Solution:
    def removeDuplicates(self, nums: List[int]) -> int:
        n = len(nums)
        i = 0
        for j in range(n):
            # 如果当前元素与nums[i]不同，将其赋值给nums[i+1]，并将i后移一位
            if nums[j] != nums[i]:
                i += 1
                nums[i] = nums[j]
        # 返回删除重复元素后数组的新长度
        return i + 1
