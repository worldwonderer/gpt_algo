Fizz Buzz

标签: 数学 字符串 模拟

难度: Easy

给你一个整数 n ，找出从 1 到 n 各个整数的 Fizz Buzz 表示，并用字符串数组 answer（下标从 1 开始）返回结果，其中：

  • answer[i] == "FizzBuzz" 如果 i 同时是 3 和 5 的倍数。
  • answer[i] == "Fizz" 如果 i 是 3 的倍数。
  • answer[i] == "Buzz" 如果 i 是 5 的倍数。
  • answer[i] == i （以字符串形式）如果上述条件全不满足。

示例 1：

输入：n = 3
输出：["1","2","Fizz"]

示例 2：

输入：n = 5
输出：["1","2","Fizz","4","Buzz"]

示例 3：

输入：n = 15
输出：["1","2","Fizz","4","Buzz","Fizz","7","8","Fizz","Buzz","11","Fizz","13","14","FizzBuzz"]

提示：

  • 1 <= n <= 104

Submission

运行时间: 20 ms

内存: 17.4 MB

def fizzBuzz(n):
    result = []
    for i in range(1, n+1):
        if i % 3 == 0 and i % 5 == 0:
            result.append("FizzBuzz")
        elif i % 3 == 0:
            result.append("Fizz")
        elif i % 5 == 0:
            result.append("Buzz")
        else:
            result.append(str(i))
    return result


class Solution:
    def fizzBuzz(self, n: int) -> List[str]:
        result = fizzBuzz(n)
        return result

Explain

此题解采用了直接遍历的方法。对于给定的整数n，我们从1遍历到n，对于每个整数i，我们检查它是否能被3和5同时整除、只能被3整除、只能被5整除，或者都不能整除。根据这些条件，我们将相应的字符串（'FizzBuzz'、'Fizz'、'Buzz'或者数字本身的字符串形式）添加到结果数组中。

时间复杂度: O(n)

空间复杂度: O(n)

def fizzBuzz(n):
    result = []  # 结果数组
    for i in range(1, n+1):  # 遍历1到n
        if i % 3 == 0 and i % 5 == 0:  # 同时是3和5的倍数
            result.append('FizzBuzz')
        elif i % 3 == 0:  # 是3的倍数
            result.append('Fizz')
        elif i % 5 == 0:  # 是5的倍数
            result.append('Buzz')
        else:  # 都不是
            result.append(str(i))  # 转换为字符串添加到结果数组
    return result

class Solution:
    def fizzBuzz(self, n: int) -> List[str]:
        result = fizzBuzz(n)  # 调用fizzBuzz函数
        return result

Explore

使用数组来存储结果而不是直接打印出来的原因是为了增加程序的灵活性和可测试性。通过将结果存储在数组中，可以方便地进行进一步的处理、测试或者验证，而且更容易重用和整合到其他系统或框架中。此外，这也使得函数更纯净（没有副作用如I/O操作），更符合函数式编程原则。

这种解法的时间复杂度为O(n)，意味着算法执行时间与输入的n成正比。如果n非常大，如一百万，算法需要逐一检查每个数字并进行条件判断，因此会消耗相应的时间。虽然对于现代计算机来说处理一百万次迭代通常是可接受的，但如果n的数量级进一步增大，性能可能会成为问题。内存使用也会随着结果数组的大小线性增加，这可能导致大量内存消耗。

使用`%`操作符来检查数字能否被3或5整除是因为这是一种简单且有效的方法来确定整数除法的余数。如果一个数i除以3或5的余数为0，那么这个数就能被3或5整除。这种方法直接、清晰且在编程中普遍使用，没有额外的性能负担，并且可以很容易地扩展到其他整除判断，使其成为处理此类问题的首选方式。

为了支持更多的条件（比如同时被3、5和7整除），可以通过增加额外的条件判断来调整函数。例如，可以添加一个条件`if i % 3 == 0 and i % 5 == 0 and i % 7 == 0`来检查是否同时满足这三个条件。为了保持代码的可读性，建议使用函数或映射来处理这些逻辑。例如，可以定义一个辅助函数或使用字典来映射条件和相应的输出。这样可以避免过多的嵌套条件语句，使得代码更加整洁和易于管理。
