不同的二叉搜索树 II 标签: 树 二叉搜索树 动态规划 回溯 二叉树    难度: Medium

Submission

# Definition for a binary tree node.
# class TreeNode:
#     def __init__(self, val=0, left=None, right=None):
#         self.val = val
#         self.left = left
#         self.right = right
class Solution:
    def generateTrees(self, n: int) -> List[Optional[TreeNode]]:
        if n < 1:
            return []
        def helper(start, end):
            l = []
            if start > end:
                l.append(None)
                return l
            
            for i in range(start, end+1):
                lefts = helper(start, i-1)
                rights = helper(i+1, end)

                for left in lefts:
                    for right in rights:
                        root = TreeNode(i)
                        root.left = left
                        root.right = right
                        l.append(root)
            return l
        return helper(1, n)

运行时间: 60 ms

内存: 16.8 MB

Explain

这个题解使用递归的方法来生成所有可能的二叉搜索树。主要思路是：对于从1到n的每个数i，将其作为根节点，然后递归生成所有可能的左子树（由小于i的数组成）和右子树（由大于i的数组成），最后将左右子树与根节点i组合起来，形成一棵完整的二叉搜索树。通过遍历1到n的每个数作为根节点，可以生成所有可能的二叉搜索树。

时间复杂度: O(4^n / n^(3/2))

空间复杂度: O(4^n / n^(3/2))

# Definition for a binary tree node.
# class TreeNode:
#     def __init__(self, val=0, left=None, right=None):
#         self.val = val
#         self.left = left
#         self.right = right
class Solution:
    def generateTrees(self, n: int) -> List[Optional[TreeNode]]:
        if n < 1:
            return []
        
        def helper(start, end):
            l = []
            # 递归的出口：当 start > end 时，返回 [None]
            if start > end:
                l.append(None)
                return l
            
            # 遍历从 start 到 end 的每个数 i 作为根节点
            for i in range(start, end+1):
                # 递归生成左子树
                lefts = helper(start, i-1)
                # 递归生成右子树
                rights = helper(i+1, end)

                # 将左右子树与根节点 i 组合成一棵完整的二叉搜索树
                for left in lefts:
                    for right in rights:
                        root = TreeNode(i)
                        root.left = left
                        root.right = right
                        l.append(root)
            return l
        
        return helper(1, n)
