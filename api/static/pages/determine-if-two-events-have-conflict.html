判断两个事件是否存在冲突 标签: 数组 字符串    难度: Easy

Submission

class Solution:
    def haveConflict(self, event1: List[str], event2: List[str]) -> bool:
        temp=[event1, event2]
        temp.sort()
        if temp[0][1]>=temp[1][0]:
            return True
        return False

运行时间: 24 ms

内存: 0.0 MB

Explain

该题解的思路是先将两个事件按照开始时间排序，然后比较第一个事件的结束时间和第二个事件的开始时间。如果第一个事件的结束时间大于或等于第二个事件的开始时间，则说明两个事件存在交集，即存在冲突，返回True。否则，返回False。

时间复杂度: O(1)

空间复杂度: O(1)

class Solution:
    def haveConflict(self, event1: List[str], event2: List[str]) -> bool:
        temp=[event1, event2]  # 将两个事件放入一个列表中
        temp.sort()  # 对事件按照开始时间进行排序
        if temp[0][1]>=temp[1][0]:  # 比较第一个事件的结束时间和第二个事件的开始时间
            return True  # 存在冲突
        return False  # 不存在冲突
