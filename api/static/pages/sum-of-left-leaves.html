左叶子之和 标签: 树 深度优先搜索 广度优先搜索 二叉树    难度: Easy

Submission

# Definition for a binary tree node.
# class TreeNode:
#     def __init__(self, x):
#         self.val = x
#         self.left = None
#         self.right = None
from collections import deque


class Solution:
    def sumOfLeftLeaves(self, root: TreeNode) -> int:
        if root is None:
            return 0

        q = deque()
        q.append((root, False))
        s = 0

        while q:
            size = len(q)
            for _ in range(size):
                node, is_left = q.popleft()
                if node:
                    if is_left and node.left is None and node.right is None:
                        s += node.val
                    q.append((node.left, True))
                    q.append((node.right, False))
        return s

运行时间: 40 ms

内存: 15.2 MB

Explain

这个题解使用广度优先搜索（BFS）的方法来遍历二叉树。它维护一个队列，将每个节点和一个布尔值（表示该节点是否为左子节点）作为元组放入队列中。在BFS的过程中，如果遇到一个节点是左叶子（即它是左子节点且没有左右子节点），就将它的值累加到变量s中。最后返回s作为所有左叶子之和。

时间复杂度: O(n)

空间复杂度: O(n)

# Definition for a binary tree node.
# class TreeNode:
#     def __init__(self, x):
#         self.val = x
#         self.left = None
#         self.right = None
from collections import deque


class Solution:
    def sumOfLeftLeaves(self, root: TreeNode) -> int:
        if root is None:
            return 0
        
        q = deque()
        q.append((root, False))  # 根节点不是左节点，初始为False
        s = 0
        
        while q:
            size = len(q)
            for _ in range(size):
                node, is_left = q.popleft()
                if node:
                    # 如果当前节点是左叶子，累加它的值
                    if is_left and node.left is None and node.right is None:
                        s += node.val
                    # 左子节点入队，并标记为左节点
                    q.append((node.left, True))
                    # 右子节点入队，并标记为非左节点
                    q.append((node.right, False))
        return s
