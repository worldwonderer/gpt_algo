删除链表中的节点 标签: 链表    难度: Medium

Submission

# Definition for singly-linked list.
# class ListNode:
#     def __init__(self, x):
#         self.val = x
#         self.next = None

class Solution:
    def deleteNode(self, node):
        """
        :type node: ListNode
        :rtype: void Do not return anything, modify node in-place instead.
        """
        node.val = node.next.val
        node.next = node.next.next

运行时间: 40 ms

内存: 15.3 MB

Explain

这个题解的思路是直接将要删除节点的值替换为下一个节点的值，然后删除下一个节点。这样可以在无法访问头节点的情况下，通过修改要删除节点的值和next指针来达到删除节点的目的。

时间复杂度: O(1)

空间复杂度: O(1)

# Definition for singly-linked list.
# class ListNode:
#     def __init__(self, x):
#         self.val = x
#         self.next = None

class Solution:
    def deleteNode(self, node):
        """
        :type node: ListNode
        :rtype: void Do not return anything, modify node in-place instead.
        """
        # 将要删除节点的值替换为下一个节点的值
        node.val = node.next.val
        # 将要删除节点的next指针指向下下个节点，跳过下一个节点，相当于删除了下一个节点
        node.next = node.next.next
