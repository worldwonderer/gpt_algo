最小化数对的最大差值 标签: 贪心 数组 二分查找    难度: Medium

Submission

class Solution:
    def minimizeMax(self, nums: List[int], p: int) -> int:
        nums.sort()
        diff = []
        if p == 0:
            return 0
        for i in range(1,len(nums)):
            diff.append(nums[i]-nums[i-1])
        def check(upper):
            i = 0
            count = 0
            while i < len(diff):
                if diff[i] <= upper:
                    count += 1
                    i += 2
                else:
                    i += 1
            return count >= p
        ans = bisect_left(range(min(diff),max(diff)+1),True,key = check) + min(diff)
        return ans

运行时间: 205 ms

内存: 30.4 MB

Explain

首先将数组排序，然后计算相邻元素的差值存入diff数组。如果p为0，则直接返回0。定义一个检查函数check，用于检查是否能找到p个差值小于等于upper的数对。使用二分查找在diff的最小值和最大值之间寻找满足条件的最小upper值，并返回该值加上diff的最小值作为结果。

时间复杂度: O(nlogn)

空间复杂度: O(n)
