按分隔符拆分字符串 标签: 数组 字符串    难度: Easy

Submission

class Solution:
    def splitWordsBySeparator(self, words: List[str], separator: str) -> List[str]:
        return [s for w in words for s in w.split(separator) if s]

运行时间: 19 ms

内存: 16.1 MB

Explain

题解采用了列表推导式来处理问题，通过两层循环来实现。首先，外层循环遍历输入的字符串数组 `words`，内层循环则是对每个字符串使用 `split` 方法按照 `separator` 进行拆分。`split` 方法会返回一个列表，其中包含分割后的子字符串。如果分割后的子字符串非空（即 `if s` 部分），则将其包含在最终的结果列表中。这种方法直接利用 Python 的内建字符串处理功能，简洁且高效。

时间复杂度: O(L)

空间复杂度: O(L)

class Solution:
    def splitWordsBySeparator(self, words: List[str], separator: str) -> List[str]:
        # 列表推导式，两层循环：外层遍历每个字符串，内层处理字符串分割
        # w.split(separator) 生成一个按 separator 分割的字符串列表
        # [s for s in w.split(separator) if s] 生成一个去除空字符串的列表
        # 最终的列表是所有这些列表的合并
        return [s for w in words for s in w.split(separator) if s]
