有重复字符串的排列组合 标签: 字符串 回溯    难度: Medium

Submission

class Solution:
    def permutation(self, S: str) -> List[str]:
        n=len(S)
        if n==0:
            return [""]
        res=[]
        for i in range(n):
            if S[i] in S[:i]:   #只需判断S[i]是否在S[:i]中出现过即可
                continue
            for s1 in self.permutation(S[:i]+S[i+1:]):
                res.append(S[i]+s1)
        return res

运行时间: 22 ms

内存: 16.0 MB

Explain

此题解采用了递归回溯的方法来生成字符串的所有排列组合。首先，对给定字符串的每一个字符，将其作为当前排列的第一个字符，然后对剩余的字符再进行排列。为了避免重复的排列，通过检查当前字符是否在它前面的字符中出现过来跳过重复的情况。这样可以确保每个字符在每个位置上只被使用一次。每一层递归都会减少一个字符，直到字符串为空，此时返回包含空字符串的数组。

时间复杂度: O(n * n!)

空间复杂度: O(n!)

class Solution:
    def permutation(self, S: str) -> List[str]:
        n=len(S)
        if n==0:
            return ['']
        res=[]
        for i in range(n):
            # 避免重复处理相同的字符
            if S[i] in S[:i]:
                continue
            # 递归处理剩余部分
            for s1 in self.permutation(S[:i]+S[i+1:]):
                res.append(S[i]+s1)
        return res
