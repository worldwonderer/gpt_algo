逆波兰表达式求值 标签: 栈 数组 数学    难度: Medium

Submission

class Solution:
    def evalRPN(self, tokens: List[str]) -> int:
        stack = []

        for x in tokens:
            if x!='+' and x!='-' and x!='*' and x!='/':
                stack.append(int(x))
            else:
                num2 = stack.pop()
                num1 = stack.pop()
                if x=='+':
                    tmp = num1+num2
                elif x=='-':
                    tmp = num1-num2
                elif x=='*':
                    tmp = num1*num2
                elif x=='/':
                    tmp = int(num1/num2)
                stack.append(tmp)
            #print(stack)
        return stack[0]

Explain

这个题解使用了栈来求解逆波兰表达式。具体思路如下：遍历输入的 tokens 列表，如果当前元素是数字，就将其转换为整数后压入栈中；如果当前元素是运算符（+、-、*、/），就从栈顶弹出两个元素，按照该运算符进行计算，并将计算结果压回栈中。遍历完整个列表后，栈顶元素即为最终的计算结果。

时间复杂度: O(n)

空间复杂度: O(n)

class Solution:
    def evalRPN(self, tokens: List[str]) -> int:
        stack = []

        for x in tokens:
            # 如果 x 是数字，将其转换为整数后压入栈中
            if x!='+' and x!='-' and x!='*' and x!='/':
                stack.append(int(x))
            # 如果 x 是运算符，从栈顶弹出两个元素，按照运算符进行计算，并将结果压回栈中
            else:
                num2 = stack.pop()
                num1 = stack.pop()
                if x=='+':
                    tmp = num1+num2
                elif x=='-':
                    tmp = num1-num2
                elif x=='*':
                    tmp = num1*num2
                elif x=='/':
                    tmp = int(num1/num2)
                stack.append(tmp)
        # 遍历完成后，栈顶元素即为最终结果
        return stack[0]
