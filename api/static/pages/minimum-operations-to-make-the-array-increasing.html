最少操作使数组递增 标签: 贪心 数组    难度: Easy

Submission

class Solution:
    def minOperations(self, nums: List[int]) -> int:
        prev = nums[0]
        count = 0
        for i in range(1, len(nums)):
            if nums[i] <= prev:
                newNum = prev + 1
                count += (newNum - nums[i])
                nums[i] = newNum
            prev = nums[i]
        return count

运行时间: 29 ms

内存: 16.6 MB

Explain

本题解采用一种贪心算法的思路。从数组的第二个元素开始遍历，对于每个元素，检查它是否大于前一个元素。如果不是，则需要将其增加到比前一个元素大1的值，同时累加所需的操作次数。通过逐个调整，确保数组的每个后续元素都比前一个大，从而使数组严格递增。

时间复杂度: O(n)

空间复杂度: O(1)

class Solution:
    def minOperations(self, nums: List[int]) -> int:
        prev = nums[0]  # 初始化prev为数组的第一个元素
        count = 0  # 初始化操作次数为0
        for i in range(1, len(nums)):
            if nums[i] <= prev:
                newNum = prev + 1  # 计算当前元素需要增加到的目标值
                count += (newNum - nums[i])  # 累加到总操作次数
                nums[i] = newNum  # 更新当前元素值
            prev = nums[i]  # 更新prev为当前元素，以便下一次比较
        return count  # 返回总操作次数
